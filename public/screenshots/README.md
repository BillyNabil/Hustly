# Screenshots

This folder contains application screenshots for documentation purposes.

## How to add screenshots

1. Run the application: `npm run dev`
2. Navigate to different pages
3. Take screenshots using your OS screenshot tool or browser DevTools
4. Save them here with descriptive names:
   - `01-landing.png` - Landing page
   - `02-dashboard.png` - Main dashboard
   - `03-kanban.png` - Ideas/Kanban board
   - `04-finance.png` - Finance tracker
   - `05-goals.png` - Weekly goals
   - `06-habits.png` - Habits tracker
   - `07-chat.png` - Ghost CEO AI Chat
   - `08-vision.png` - Vision board
   - `09-analytics.png` - Analytics dashboard
   - `10-leaderboard.png` - Leaderboard

## Screenshot Guidelines

- Use **1920x1080** resolution (or proportional)
- Ensure **dark mode** is enabled
- Capture **full page** when possible
- Include **realistic data** (not placeholder)
- **Clean browser chrome** (hide bookmarks bar, extensions)
